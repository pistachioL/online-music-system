# 海纳音乐播放器
该项目是用Vue实现的前端界面，后台数据要配合启动该项目：https://github.com/pistachioL/Music-Web-Crawler


### Project setup
```
npm install
```

Compiles and hot-reloads for development
```
npm run serve
```

Compiles and minifies for production
```
npm run build
```

Lints and fixes files
```
npm run lint
```
